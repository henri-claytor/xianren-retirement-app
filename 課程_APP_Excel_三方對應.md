### CH1 現在的你，離退休有多遠？

| 課程子題 | APP 功能 | Excel Sheet | 落差評估 |

## 三、落差統計摘要

### 🟢 APP 獨家（Excel 沒有）
1. Monte Carlo 壓力測試（A2）
2. 三風險分項敏感度（A2）
3. 風險屬性一鍵選擇器（A3）
4. 核心衛星策略 + 投資工具比較（A3）
5. 再平衡規則 R1/R2/R3（B3）+ 再平衡建議（B4）
6. 穩定現金流覆蓋率三階段識別（B2）
7. Dashboard 快速試算滑桿
8. 退休生活內容/社群（C1/C2）

### 🟡 Excel 優勢（APP 較弱或沒有）
1. 12 個月家計簿逐月追蹤（表1）→ APP 僅快照輸入
2. 多年費用預估含年增率（表3）→ APP 只能即時試算
3. 年度投報率歷史記錄（表7）→ APP 僅即時快照
4. 退休後資產負債表（表9）→ APP 無專屬頁

### 🔴 共同缺口（課程有，兩方都沒有）
1. CH5 家庭照顧預算模組
2. CH5 長照保險試算
3. CH5 父母醫療緊急預備金

### ✅ 三方完全對應
1. 資產/負債盤點
2. 退休金目標計算
3. 通膨對支出的影響
4. 提領率試算
5. 退休後逐年現金流

## 四、課程教學搭配建議

### 每一章的工具選擇原則

| 情境 | 推薦工具 | 原因 |
|------|---------|------|
| 課堂即時示範 | **APP** | 互動性強、視覺化即時 |
| 學員自行深入練習 | **Excel** | 格式彈性、適合有財務基礎的學員 |
| 跨年度長期追蹤 | **Excel** | 多年資料按格存取較直覺 |
| 情境試算（滑桿/壓力測試） | **APP** | Monte Carlo、滑桿是 Excel 難做 |
| 收入結構拆解（勞保/勞退/租金/股利） | **Excel 表4** | 獨立 sheet 更清楚 |
| 三桶金/再平衡 | **APP** | CH4 核心，APP 有完整閉環 |

### 建議課程配置

```
CH1 →  APP Dashboard + S1   |  Excel 表1、表5
CH2 →  APP A1、A2           |  Excel 表3、表4、表6
CH3 →  APP A3               |  Excel 表5、表6、表7
CH4 →  APP S2、B1、B2、B3、B4 |  Excel 表8、表9
CH5 →  （缺口，待補）         |  （缺口，待補）
CH6 →  APP C1、C2           |  —
總覽 → APP Dashboard         |  Excel 表10
```

## 五、給嫺人的對接建議

1. **APP 可涵蓋的 Excel 核心功能**：表1~表6 的計算類 sheet，APP 已有對應，學員可擇一使用
2. **Excel 仍具優勢的場景**：多年歷史追蹤（表3、表7、表9）— 建議作為課程「年度檢視」作業模板
3. **課程中 APP 獨家教學點**：Monte Carlo、三風險敏感度、風險屬性、再平衡規則、穩定覆蓋率 — 這些是 Excel 做不到的，是 APP 的教學亮點
4. **待補強方向**：CH5 家庭照顧模組是 APP 與 Excel 的共同缺口，未來可同步擴充
