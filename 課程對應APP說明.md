| 章節 | 課程主題 | APP 主要頁面 | 對應 Stage |
|------|---------|-------------|-----------|
| CH1 | 現在的你，離退休有多遠？ | Dashboard ScenarioSummary + S1 | 現況盤點 |
| CH2 | 從現況到缺口：認識你的退休風險 | A1 / A2 | 缺口診斷 |
| CH3 | 建立你的投資判斷力 | A3 | 資產配置 |

| CH5 | 退休之外，你還要照顧誰？ | （尚未納入，規劃中）| 家庭照顧 |

## CH1 — 現在的你，離退休有多遠？

**課程目標**：讓學員盤點現況、對退休時程有量化感受。

### 對應功能

| 功能 | 頁面 | 說明 |
|------|------|------|
| 情境摘要 | Dashboard / ScenarioSummary | 依 S1 資料自動生成「距退休 X 年」的敘述，分三種措辭：近期（<5）/ 中期（5–15）/ 遠期（>15） |
| 快速編輯 | ScenarioSummary Accordion | 點右上鉛筆圖示可直接修改退休年齡與退休月支出，無需進入 S1 |
| 現況盤點 | S1 財務輸入 | 收入、支出、資產、負債、退休目標等基本資料 |
| 退休缺口卡 | Dashboard VerdictCard | 顯示距退休年數、初估缺口等關鍵數字 |

**設計原則**：零問卷。以編輯既有資料取代填表，降低使用者動作。

---

## CH2 — 從現況到缺口：認識你的退休風險

**課程目標**：讓學員理解通膨、長壽、醫療三大風險對退休成功率的影響。

### 對應功能

| 功能 | 頁面 | 說明 |
|------|------|------|
| 退休金目標 | A1 RetirementGoal | 計算達成目標所需退休金總額 |
| 通膨影響區塊 | A1 | 呈現「今日月支出 vs 退休時月支出（通膨調整後）」對比 |
| 壓力測試 | A2 StressTest | 蒙地卡羅模擬 1000 次，計算成功率 |
| 三風險分項敏感度 | A2 結果頁 | 三張卡片（通膨 +1% / 長壽 +5 年 / 醫療支出 +20%），各跑 300 次 Monte Carlo |
| 最需留意標籤 | A2 | 差值最負的卡片紅框 + 「最需留意」警示 |

**設計原則**：單因子敏感度，在結果頁附上免責說明。

---

## CH3 — 建立你的投資判斷力

**課程目標**：讓學員建立風險屬性、了解核心衛星策略與投資工具差異。

### 對應功能

| 功能 | 頁面 | 說明 |
|------|------|------|
| 風險屬性選擇 | A3 RiskProfileSelector | 三張橫向卡（保守 / 穩健 / 積極），一鍵套用三桶金 preset，取代傳統問卷 |
| 三桶金配置 | A3 AssetAllocation | 短期 / 中期 / 長期桶比例設定 |
| 核心衛星策略 | A3 說明區塊 | 雙卡片說明核心 70% / 衛星 30% 的資產分工 |
| 投資工具比較 | A3 比較表 | ETF / 共同基金 / 個股 / 高股息，四維度比較（流動性、費用、風險、適合族群） |

**風險 preset 對應**：
- 保守：短 20% / 中 40% / 長 40%
- 穩健：短 15% / 中 35% / 長 50%
- 積極：短 5% / 中 15% / 長 80%

---

## CH4 — 讓錢穩定流進來：三桶金策略

**課程目標**：讓學員建立三桶金、提領規則、再平衡機制。

### 對應功能

| 功能 | 頁面 | 說明 |
|------|------|------|
| 三桶金總覽 | S2 BucketOverview | 目前 vs 建議比例，依距退休年數動態推薦（取代固定 10/30/60） |
| 提領計畫 | B1 WithdrawalPlan | 年度提領金額、提領率設定 |
| 現金流時間軸 | B2 CashflowTimeline | 逐年現金流、資產餘額視覺化 |
| 再平衡規則 | B3 RebalanceRules | 三條規則：R1 市場跌幅 / R2 短桶警戒 / R3 年度定期 |
| 再平衡建議 | B4 Rebalancing | 依規則輸出具體調整動作 |
| 短桶警戒 | S2 → B3 | 短桶可撐月數 < 門檻時紅色警戒，點擊連至 B3 R2 |

**規則詳情**：
- R1：市場跌幅 >20% 觸發，人工判斷
- R2：短桶金額 ÷ 月支出 < 門檻時自動警戒（門檻可調）
- R3：每年定期（依當前月份提示）
