## Why

本 change 的目標是產出一個**前端示範網頁（Web Prototype）**，供作者嫺人確認 A1~A4 退休前規劃工具的內容、計算邏輯與互動設計，取代現有 Google Sheet，作為後續正式 APP 開發的討論基礎。

所有計算在前端執行，資料以 localStorage 儲存，不需要後端 API 或資料庫。待作者確認內容正確後，再規劃正式後端實作（10,000 次 Monte Carlo、後端 API、原生 APP）。

## What Changes

- **新增** A1 退休目標設定工具：輸入退休期望參數，自動計算退休金目標金額與缺口
- **新增** A2 壓力測試工具：蒙地卡羅模擬（10,000次）計算退休成功率，支援多種壓力情境分析
- **新增** A3 資產配置建議工具：依壓力測試結果給出三桶金目標配置、再平衡建議與標的清單
- **新增** A4 定期追蹤更新工具：定期快照財務現況，追蹤退休成功率歷史趨勢，支援歸桶異動通知
- **共用** S1 財務現況輸入：支援台股/美股/基金標的搜尋與即時報價，支援手動輸入現金/不動產/負債
- **共用** S2 三桶金現況總覽：依歸桶規則（股票→長期、ETF/基金依債券比例≥55%→中期）自動分類
- **共用** S3 通膨侵蝕模擬器：視覺化通膨與長壽對資產的侵蝕曲線

## Capabilities

### New Capabilities

- `financial-input`: 財務現況輸入模組，涵蓋資產（台股/美股/基金即時報價）、收入、支出三分類、負債輸入，串接公司財經資料 API
- `three-bucket-overview`: 三桶金現況總覽，依歸桶規則自動分類並視覺化呈現，支援手動覆蓋與歷史比較
- `inflation-simulator`: 通膨侵蝕模擬器，視覺化多情境資產遞減曲線
- `retirement-goal-calculator`: 退休金目標試算機，含通膨調整、4%/3%提領法則、年金終值計算
- `stress-test`: 蒙地卡羅壓力測試，10,000次模擬，輸出成功率分布、PR75/50/25/5資產走勢、系統建議
- `asset-allocation`: 資產配置建議，三桶金目標配置計算、再平衡建議、標的建議清單
- `periodic-tracking`: 定期追蹤快照，歷史趨勢圖、歸桶異動通知、推播提醒

### Modified Capabilities

（無，本次為全新模組）

## Impact

- **產出**：可在瀏覽器直接執行的靜態前端網站（Vite + React + TypeScript）
- **資料儲存**：localStorage（無後端依賴）
- **計算方式**：所有計算在前端執行（Monte Carlo 簡化為 1,000 次）
- **部署方式**：`npm run build` 產出靜態檔，可上傳至 Vercel / GitHub Pages 供作者操作
- **後續**：作者確認後，再依 tasks.md 規劃正式後端 API、10,000 次模擬、原生 APP 實作
